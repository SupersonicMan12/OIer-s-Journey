BST = Binary Search Tree.

1. Empty is BST
2. Each node has at most two children, one left one right
3. If left not empty, all values in left child subtree < this node
4. If right not empty, all values in right child subtree >(=) this node
5. Both left and right subtrees are BST

Operations' efficiency is based on depth of tree. If degenerate case of chain of n, O(n) per op.
